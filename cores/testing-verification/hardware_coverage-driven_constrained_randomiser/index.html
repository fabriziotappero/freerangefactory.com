Description

This project aims to create a synthesisable coverage-driven constrained randomisation testbench library. The core uses the hardware TLM approach for bus interfaces between the tester and the DUT. Many concepts used in this project are similar to the OS-VVM framework, such as the intelligent coverage methodology, only that we have made them synthesisable to hardware so this library can be reused as hardware testers in hardware testbench environments.
