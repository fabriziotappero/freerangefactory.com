Description

This project is based on the OR1K2 SoC.
It is enhanced using System Hyper Pipelining (SHP) technology. You can achieve a higher performance on a given area and you can run up to 16 different OR1K2 cores at the same time.

Schematic

Board

It is mapped on a Low Cost FPGA Arduino board. The board has a Spartan 6, (LX16), HS-USB interface, 2 independent SDRAM interfaces to 512MBit / 64MBit SDRAM at 166MHz.
