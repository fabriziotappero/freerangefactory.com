Description

This code implements a Reed Solomon encoder/decoder for both the parity size of 8 and the 16.
The Reed Solomon encoder structure implemented here is the one used in ADSL G.9956 but it is
also used in lot of the other emerging communication systems such as wireless 3rd generation
3GPP standard (Wideband CDMA) and powerline communication standards like Homeplug.
The code is not documented very well. I found it on the internet and tested it and
find that it works very very good. Use and enjoy by saving lot of time implementing this big block.
This code can also be used for verilog coding since is written is such as way to be easy
to port it to the behavioral verilog from the C language.
