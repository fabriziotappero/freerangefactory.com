Intro

Zopfli Compression Algorithm is a new zlib (gzip, deflate) compatible compressor.
The output compresses by Zopfli is typically around 5% better (smaller) than zlib (at maximum compression) and better than any other zlib-compatible compressor we have found.
Because software implementation of this algorithm takes ~100x slower then zlib, need creation of hardware implementation Zopfli compressor coprocessor.
It is a compression-only implementation; existing tools can decompress the data. Zopfli is bit-stream compatible with compression used in gzip, zip, png, etc.
