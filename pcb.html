DESIGN A PRINTED CIRCUIT BOARD

Learn how to design and make an electronics printed circuit board is an essential skill for anyone interested in electronics. The following short tutorials will teach you how to use the software tool Eagle to make your own PCB board.

THE CONTENT OF THESE TUTORIALS IS TAKEN FROM THE WEBSITE sparkfun.com AND NO CONTENT OWNERSHIP IS THEREFORE CLAIMED.

Now you are ready for some PCB action. So that you can start ahead do not forget to download few important things like the Sparkfun Eagle library, the Sparkfun setup script and check out the Sparkfun Gitub repo. Also, in case you need to generate a GERBER fileset, please use the Sparkfun gerb274x job file.
